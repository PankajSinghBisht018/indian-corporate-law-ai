AI LEGAL ASSISTANT - SETUP GUIDE

1. SYSTEM REQUIREMENTS
- Python 3.9+
- macOS / Linux / Windows
- Internet connection (for first-time setup)

--------------------------------------------------

2. CLONE PROJECT

git clone <your-repo-link>
cd ai-legal-assistant

--------------------------------------------------

3. CREATE VIRTUAL ENVIRONMENT

python3 -m venv venv

Activate:

Mac/Linux:
source venv/bin/activate

Windows:
venv\Scripts\activate

--------------------------------------------------

4. INSTALL DEPENDENCIES

pip install -r requirements.txt

--------------------------------------------------

5. INSTALL OLLAMA (LLM ENGINE)

Mac/Linux:
curl -fsSL https://ollama.com/install.sh | sh

OR download from:
https://ollama.com/download

Verify:
ollama --version

--------------------------------------------------

6. DOWNLOAD MODEL

Recommended (fast + stable):

ollama pull mistral

Optional:
ollama pull phi3
ollama pull llama3
ollama pull deepseek-r1

--------------------------------------------------

7. RUN PROJECT

python main.py

--------------------------------------------------

8. EXAMPLE QUERY

What is Section 247?

--------------------------------------------------

COMMON ERRORS & FIXES

1. ModuleNotFoundError
Cause: Missing libraries

Fix:
pip install -r requirements.txt

--------------------------------------------------

2. ollama: command not found
Cause: Ollama not installed or PATH issue

Fix:
Reinstall Ollama OR restart terminal

--------------------------------------------------

3. model 'mistral' not found
Cause: Model not downloaded

Fix:
ollama pull mistral

--------------------------------------------------

4. Slow response
Cause: Using heavy model (deepseek-r1)

Fix:
Use mistral or phi3

--------------------------------------------------

5. Chinese output
Cause: DeepSeek default language

Fix:
Use mistral OR update prompt to force English

--------------------------------------------------

6. search not defined error
Cause: Wrong function placement

Fix:
Ensure all functions are above main block

--------------------------------------------------

7. numpy / faiss not found
Cause: venv not activated

Fix:
source venv/bin/activate
pip install -r requirements.txt

--------------------------------------------------

9. PERFORMANCE TIPS

- Use model='mistral' for best speed
- Keep top_k=2 for faster answers
- Do not rebuild embeddings every run

--------------------------------------------------

10. FINAL NOTE

This project uses:
- FAISS for retrieval
- Sentence Transformers for embeddings
- Ollama + Mistral for LLM

--------------------------------------------------
